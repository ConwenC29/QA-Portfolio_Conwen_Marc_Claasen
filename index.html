About Me

I am a Software Quality Assurance Analyst with over 12 years of experience in testing across desktop, web, and mobile applications. My focus is on ensuring software quality through structured testing approaches, strong attention to detail, and a user-first mindset.

I have hands-on experience in both manual and API testing, as well as database validation using SQL Server. My work includes functional testing, regression testing, exploratory testing, and supporting User Acceptance Testing (UAT) cycles.

I also have experience using tools such as Jira, Zephyr Scale, and Confluence for test management and defect tracking. I am actively expanding my skills in test automation using Playwright with Python to strengthen my automation capabilities.

I am passionate about improving software quality not only by identifying defects, but by contributing to better design, usability, and overall user experience.
